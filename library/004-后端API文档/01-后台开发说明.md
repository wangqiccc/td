# 后台接口开发说明

### 技术说明
| 技术类型 | 版本 |
|-----------|---------------|
| PHP      |  -             |
| Mysql    |  -             |
| Apache   |  -             |
| ThinkPHP |  5.0           |

### RestFull接口规范
| 请求类型 | 说明 |
|---------|-------------|
| get     | 获取 |
| post    | 添加 |
| put     | 修改 |
| delete  | 删除 |

### 返回状态码说明
| 状态码 | 说明 |
|---------|-------------|
| 200     | 请求已成功，请求所需数据体将随此响应返回 |
| 404     | 请求失败，请求所希望得到的资源未被在服务器上发现 |
