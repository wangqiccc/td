## 获取政策（政策法规/服务事项/办理指导）


### 接口详情
### 请求地址

/api/policy/policy 

### 请求类型
GET

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| ----- | :---: | :---: | --- | --- | --- |
| pid   | number | 否 | 表示政策类型 | -  | 1 |
| id  | number | 否 | 表示政策ID | - | 1 |
| name | string | 否 | 按照政策名称进行搜索 | - | 生产 |
| page	| number | 否 |	页数  | - | 1   |
| pageSize | number |否 |	每页展示数量 | 10 | 5 |
