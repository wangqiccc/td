## 获取动态信息


### 接口详情
### 请求地址

/api/activity/activity

### 请求类型
GET

### 请求参数
| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| ----- | :---: | :---: | --- | --- | --- |
| type | number | 是 | 要获取信息的类型 | -  | 0 |
| page | number | 否 | 页数 | - | 1 |
| pageSize | number | 否 | 每页展示数量 | 10 | 5 |
| id | number | 否 | 表示信息ID | - | 21 |
